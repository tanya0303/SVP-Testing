## Local Development

To run locally:
```bash
cd "salesforce-planning-app"
npm start
```

Local URLs will use `http://localhost:3000/flow-phoenix/...` format.

## Deployment

The app is automatically deployed to GitHub Pages. To update:
```bash
npm run deploy
```

This will rebuild and publish the latest version.
