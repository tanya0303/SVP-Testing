# Salesforce Planning App

This React application is built to match the Figma design specification for a Commercial Planning and Forecasting interface.

## Features

- Hero container with breadcrumb navigation
- Progress indicator showing completion status
- Resource cards sidebar with:
  - Video preview card
  - Latest updates information
  - Trailhead learning links

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

## Note

All buttons and interactive elements are currently static and non-functional as per requirements. Functionality will be added in future updates.

## Technology Stack

- React 18
- Pure CSS (no CSS frameworks)
- SVG icons embedded as data URIs
